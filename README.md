\# Cathedral OS



A personal operating system for intentional living, featuring daily rhythms, task management, and notes.



\## Features

\- \*\*Flow View\*\*: Daily schedule with time blocks and task tracking

\- \*\*Calendar View\*\*: Monthly calendar with task overview

\- \*\*Notes\*\*: Quick notes and reflections

\- \*\*Dark/Light Mode\*\*: Toggle theme

\- \*\*PWA Ready\*\*: Install as a progressive web app



\## Getting Started

1\. Open `index.html` in your browser

2\. Data is saved locally in your browser's localStorage

3\. Install as an app for mobile-like experience



\## Usage Tips

\- Click on time blocks to view/add tasks

\- Use the calendar to navigate between days

\- Add notes for reflections and ideas

\- Toggle theme with the sun/moon icon



\## File Structure

\- `index.html` - Main HTML structure

\- `style.css` - All styling

\- `app.js` - All JavaScript functionality

\- `README.md` - This documentation
